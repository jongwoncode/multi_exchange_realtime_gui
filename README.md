# **Bitget, Binance 거래소 실시간 Orderbook, 1m chart GUI**

## **1. 프로젝트 정보**
    
* 😀 프로젝트 구성원 : 이종원(개인프로젝트)

* 🤖 주요 사용 기술  : PyQt5, multi-processing, websoket, request, finpot

  * **🧢기술적 실력 향상**
    1. PyQt5를 사용한 GUI 프로그램 생성.

    3. Websocket을 사용하여 real-time data control하기
